Visão Geral

Este repositório reúne os estudos, atividades e projetos realizados na disciplina de Desenvolvimento Front-End para Web. O objetivo é consolidar os conhecimentos em HTML, CSS e JavaScript, além de demonstrar aplicação prática de design, estilos, responsividade e boas práticas de interface.

Objetivo da Disciplina

Aprender a estruturar páginas web usando HTML semântico.

Aplicar estilos e layouts com CSS (posicionamento, responsividade, tipografia, cores etc.).

Criar interatividade e comportamento dinâmico com JavaScript.

Desenvolver interfaces amigáveis, acessíveis e responsivas, adequadas a diferentes dispositivos e tamanhos de tela.

Promover organização de código, clareza e manutenção em projetos front-end.

Conteúdos e Conceitos Abordados
HTML

Marcação semântica usando tags adequadas (header, nav, main, section, footer etc.)

Estruturação de conteúdo: texto, listas, tabelas, links, formulários, inputs e elementos de mídia

Acessibilidade básica e boas práticas de HTML

CSS e Layout

Box model, seletores, propriedades e regras de estilo

Layouts com Flexbox e Grid

Responsividade usando media queries

Estilos visuais: tipografia, cores, espaçamento, alinhamentos

Organização de estilos (separação de estrutura, estilo e comportamento)

JavaScript e Interatividade

Manipulação do DOM (seleção de elementos, manipulação de conteúdo e classes)

Eventos (cliques, submissão de formulários, interações do usuário)

Validações de formulário e feedback para usuário

Criação de comportamento dinâmico em páginas estáticas

Boas Práticas de Front-End

Separação clara entre estrutura, estilo e comportamento

Organização de arquivos e pastas para facilitar manutenção

Código limpo e legível — para facilitar colaboração e revisões futuras

Considerações de usabilidade e acessibilidade (uso correto de tags, semântica, formulários acessíveis)
