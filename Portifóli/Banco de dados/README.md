Introdução

Este repositório reúne minha evolução na disciplina Banco de Dados, incluindo estudos teóricos, exercícios, modelagens e projetos realizados ao longo do curso. Ele registra meu processo de aprendizagem desde os primeiros conceitos até consultas mais complexas e criação de estruturas relacionais completas.

O conteúdo aqui apresentado reflete minha compreensão sobre como projetar, organizar e manipular dados de forma eficiente e segura — habilidades essenciais para desenvolvimento de software, análise de dados e sistemas de informação.

Objetivos da Disciplina

A disciplina tem como finalidade:

Desenvolver compreensão sólida sobre a função e importância dos bancos de dados.

Ensinar os principais modelos de organização de dados e suas aplicações.

Capacitar o aluno a projetar estruturas eficientes utilizando modelos conceitual, lógico e físico.

Ensinar a linguagem SQL e suas operações fundamentais.

Formar base para manipulação segura, consistente e integrada de informações.

Este repositório acompanha esse processo, mostrando a aplicação prática de cada tema.

Tópicos Estudados
1. Fundamentos de Banco de Dados

O que é um banco de dados

Diferença entre dado e informação

Importância dos SGBDs

Conceitos de integração, segurança e consistência de dados

2. Sistemas Gerenciadores de Banco de Dados (SGBDs)

Função e propósito de um SGBD

Tipos de SGBDs

Conceitos de tabelas, registros, chaves e atributos

3. Modelagem de Dados

A modelagem é um dos pilares da disciplina e envolve entender o problema, identificar entidades e organizar informações de forma estruturada.

Modelagem Conceitual

Criação do Diagrama Entidade-Relacionamento (DER)

Identificação de entidades, atributos e relacionamentos

Definição de cardinalidades

Representação das regras de negócio

Modelagem Lógica

Conversão do modelo conceitual em tabelas

Definição de chaves primárias e estrangeiras

Estabelecimento das dependências funcionais

Normalização até as formas mais utilizadas (1FN, 2FN e 3FN)

Redução de redundâncias e eliminação de anomalias

Modelagem Física

Tradução do modelo lógico para o SGBD escolhido

Criação das tabelas e seus tipos de dados

Definição de restrições de integridade

Organização final utilizada pelo banco
